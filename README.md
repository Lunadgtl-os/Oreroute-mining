# Ore Route Mining

Ore Route by Sovereign Salts is an enterprise mineral movement, custody, processing and trade-evidence platform.

## Current migration phase

This repository is the independent source-controlled version of the Ore Route platform. The first migration release provides:

- A realistic mining-led visual identity using the approved Ore Route hero asset
- Operations overview and exception desk
- Digital material passport register
- Live shipment and route-control views
- Chain-of-custody timeline
- Transport tender board
- Wash-plant batch register
- Trade dossier register
- Evidence vault
- Automated build validation
- GitHub Pages deployment after changes are merged to `main`

The current interface uses representative operational records while the Base44 data entities and authentication layer are connected to the independent application.
